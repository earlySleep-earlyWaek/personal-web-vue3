娱乐项目介绍:  
点击开始按钮后,方框区会随机位置出现随机大小的〇,〇的大小和消亡时间会随着时间发生改变.  
有独特的评分机制,不同〇有不同的积分,整体上,越往后的〇积分越高.
注:为防止后期过于逆天(指出现星空的情况),在生成速度和大小上添加了保底机制.
