## 点点点  
'点点点'介绍:  
使用你的鼠标点击哪些不断出现在屏幕中的黑色圆圈，它们会给予你一些得积分。  
黑色圆圈的出现速度会越来越快，面积也会越来越小，直到它们达到我预定的阈值（当然，你也可以决定一部分）。
