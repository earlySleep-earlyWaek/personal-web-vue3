# 小创意  
 描述：进行一些娱乐项目开发，目前已经开发的娱乐项目包括“点点点”、“Div杀手”，后续准备开发'Div大战'，'打DIV'。  
  
## 点点点  
'点点点'介绍：  
使用你的鼠标点击哪些不断出现在屏幕中的黑色圆圈，它们会给予你一些得积分。  
黑色圆圈的出现速度会越来越快，面积也会越来越小，直到它们达到我预定的阈值（当然，你也可以决定一部分）。

### Div杀手  
'Div杀手'介绍  
使用鼠标操控自己的div，除了躲避大家伙的吞噬,记得消灭那些小家伙。  
有一个永远比你强大的boss，记得远离它。
