## 点点点  
'点点点'介绍:  
通过点击出现的圆圈,获取积分.  
每个圆圈依据圆圈的大小,生命时长有对应的积分,整体上越往后积分越高.  
注:添加了圆圈大小以及生成速度的保底数值.
